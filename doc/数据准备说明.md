# 数据准备说明

## 一、必须准备的文件

### 1. 风格图像

放置位置：

```text
data/style_images/
```

建议先准备 1 张风格图，命名为：

```text
data/style_images/style.jpg
```

选择建议：

- 颜色和纹理明显。
- 笔触或图案有辨识度。
- 分辨率不需要特别大，长边 512-1024 像素即可。
- 可以选择梵高《星空》、浮世绘、莫奈、水彩、素描等风格。

### 2. 训练内容图像

放置位置：

```text
data/train_images/
```

高分建议：

- 正式训练准备 1000-5000 张自然图像。
- 如果先本地跑通流程，可先放 100-300 张。
- 图片可以来自 ImageNet 子集、COCO 子集，或其他公开自然图像数据集。
- 图片内容应尽量多样，包括建筑、风景、人物、物品等。

### 3. 测试内容图像

放置位置：

```text
data/test_images/
```

建议准备 3-5 张自己的测试图片：

- 1 张校园或建筑图。
- 1 张自然风景图。
- 1 张人物、物品或室内图。

命名示例：

```text
data/test_images/campus.jpg
data/test_images/landscape.jpg
data/test_images/object.jpg
```

## 二、训练后会生成的文件

模型权重：

```text
checkpoints/best_model.pth
```

训练日志：

```text
logs/train_loss.csv
logs/environment.txt
logs/run_commands.txt
```

实验结果：

```text
results/samples/
results/comparisons/
results/metrics.csv
results/loss_curves.png
```

## 三、当前最小启动数据

如果现在只是先跑通代码，最低需要：

```text
data/style_images/style.jpg
data/train_images/至少 20-50 张图片
data/test_images/至少 1 张图片
```

如果目标是期末高分，建议最终准备：

```text
data/style_images/style.jpg
data/train_images/1000-5000 张训练图
data/test_images/3-5 张测试图
```

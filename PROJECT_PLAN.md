## 四、技术架构

### 核心模型
```
内容图像 (Content Image)
    ↓
TransformNet (图像转换网络)
    ↓
生成图像 (Generated Image)
    ↓
VGG16 (特征提取网络)
    ↓
计算损失 (content_loss + style_loss)
    ↓
反向传播更新 TransformNet
```

### 训练参数（建议）
- **学习率**：1e-3
- **批大小**：4 或 8
- **训练轮数**：2-4 epochs
- **图像尺寸**：256×256
- **损失权重**：
  - content_weight = 1.0
  - style_weight = 1e5
  - tv_weight = 1e-6

### 数据需求
- **风格图**：1张（512×512 以上）
- **训练图**：1000-2000张（ImageNet 或 COCO）
- **测试图**：5张（自己的照片）

## 六、风险与应对

### 可能的问题
1. **数据集下载慢**：使用国内镜像或 Kaggle
2. **训练时间长**：先小规模测试，再正式训练
3. **显存不足**：调小 batch_size 或图像尺寸
4. **效果不理想**：调整 style_weight

### 保底方案
如果训练遇到严重问题：
1. 使用更小的数据集（500张）
2. 训练更少的 epoch（1-2轮）
3. 降低图像分辨率（128×128）

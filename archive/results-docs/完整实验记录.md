## 实验背景

### 任务目标
实现快速神经风格迁移（Fast Neural Style Transfer），将任意输入图像转换为梵高《星空》风格。

### 技术方案
- **模型**: TransformNet（编码器-解码器 + 5个残差块）
- **损失函数**: Perceptual Loss (Content Loss + Style Loss + TV Loss)
- **特征提取**: VGG16（预训练，frozen）
- **关键技术**: Instance Normalization, Residual Blocks, Reflection Padding

### 评价指标
1. **delta_time**: 推理时间（目标 < 0.1秒）
2. **content_loss**: 内容保留程度（越低越好）
3. **style_loss**: 风格匹配程度（越低越好）
4. **视觉效果**: 主观评价清晰度和艺术感

## 分辨率问题诊断

### 问题发现

用户反馈：**256×256 训练的模型，在 512×512 测试时效果模糊**

**原因分析**:

#### 问题1: 训练-测试尺寸不匹配
- 训练: 256×256
- 测试: 512×512
- 结果: 模型外推到未见过的尺寸，产生模糊

**验证实验**:
```bash
# 256×256 测试
python3 test.py --model checkpoints/model_15epochs.pth --input data/test_images/test.jpg --output results/output_15epochs_256.png --image-size 256
```

**结果**:
- **delta_time**: 0.107s
- **content_loss**: 24403.12
- **style_loss**: 65.60
- **输出图像**: `results/output_15epochs_256.png`

**观察**: ✅ 256×256 测试效果更清晰！证明尺寸匹配的重要性

---

#### 问题2: 图像被裁剪

**发现**:
- 原始测试图: 512×288（宽屏）
- 输出图像: 512×512（正方形）
- **左右两边被裁掉约 112 像素！**

**原因**: `utils/image.py` 使用 `Resize + CenterCrop` 强制裁剪成正方形

**代码问题**:
```python
# 原来的代码
steps.append(transforms.Resize(image_size))
steps.append(transforms.CenterCrop(image_size))  # 裁剪！
```

**解决方案**: 修改为拉伸模式（保留全部内容）

```python
# 修改后
elif resize_mode == 'stretch':
    # 拉伸到正方形（轻微变形，但保留全部内容）
    steps.append(transforms.Resize((image_size, image_size)))
```

## 总结与建议

### 核心发现

#### 1. 训练数据量
- **最少要求**: 500-1000 张图像
- **推荐**: 6000+ 张图像
- **实验证明**: 13 张 → 完全失败；6912 张 → 成功

#### 2. 训练轮数
- **快速测试**: 2-3 epochs（仅验证代码）
- **开发调试**: 5-8 epochs（看到基本效果）
- **实用部署**: 10-15 epochs（推荐）
- **追求极致**: 15-20 epochs

#### 3. 图像尺寸
- **关键原则**: 训练尺寸必须与测试尺寸匹配
- 256×256 训练 → 256×256 测试 ✅
- 256×256 训练 → 512×512 测试 ❌ 模糊
- 512×512 训练 → 512×512 测试 ✅ 清晰

#### 4. 权重平衡
- **content_weight = 1.0, style_weight = 1e5**: 标准配置
- **content_weight = 2.0, style_weight = 1e5**: 更清晰的内容
- **style_weight 调整**: 1e4（弱）、1e5（中）、1e6（强）

#### 5. 图像预处理
- **训练**: CenterCrop（正方形）
- **测试**: Stretch（保留全部内容，轻微变形）
- **避免**: 测试时裁剪导致内容丢失
